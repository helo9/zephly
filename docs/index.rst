Welcome to frickelflightcontrol's documentation!
================================================

**frickelflightcontrol** is a `Zephyr RTOS`_ based quadrotor flight controller code
aiming at broad hardware support by leveraging Zephyr as foundation. Thereby also 
older hardware is supported which is not supported anymore in popular codes.
